BV Shop 商品圖層選擇器 - 圖片資料夾說明
==========================================

【圖片規格要求】
1. 格式：PNG（需要透明背景）
2. 尺寸：建議 800x800 像素（正方形）
3. 所有圖層尺寸必須相同，才能正確對齊

【資料夾結構】

images/
├── base.png          ← 底圖（商品基礎圖，可以是不透明的）
├── spec1/            ← 規格1 的圖層（例如：車種）
│   ├── yamaha.png
│   ├── honda.png
│   ├── suzuki.png
│   └── kawasaki.png
├── spec2/            ← 規格2 的圖層（例如：外圈顏色）
│   ├── black.png
│   ├── red.png
│   ├── blue.png
│   ├── gold.png
│   └── silver.png
└── spec3/            ← 規格3 的圖層（例如：油箱蓋顏色）
    ├── black.png
    ├── red.png
    ├── blue.png
    ├── gold.png
    └── titanium.png

【圖片命名規則】
- 檔名需要與 config.js 中的 value 對應
- 例如 config.js 中設定 { value: "black", ... }
- 則圖片檔名為 black.png

【製作圖層的建議】
1. 在 Photoshop/Illustrator 中建立相同尺寸的畫布
2. 每個規格選項製作一個獨立的圖層
3. 匯出時選擇「PNG-24」並勾選「透明度」
4. 確保各圖層的位置對齊一致

【測試步驟】
1. 將圖片放入對應資料夾
2. 用瀏覽器開啟 index.html
3. 選擇不同規格，觀察圖層是否正確疊加
4. 如果圖層位置偏移，檢查圖片尺寸是否一致

【注意事項】
- 圖片檔名不要有中文或空格
- 所有圖片尺寸必須相同
- 建議先用小尺寸圖片測試，確認功能正常後再換成高解析度圖片
